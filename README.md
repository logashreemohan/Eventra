# Eventra - Comprehensive Wedding Planning Platform

## Overview
Eventra is a full-featured wedding planning platform that connects couples with verified vendors while providing comprehensive planning tools. This project has been significantly enhanced with numerous new features to create a complete wedding planning ecosystem.

## Key Features

### Core Platform
- **Customer Website**: Landing page, service listings, vendor details, checkout, login/signup, and profile management
- **Vendor Dashboard**: Vendor management, bookings, payments, and profile customization
- **Admin Panel**: User management, vendor approval, and system analytics

### Enhanced Wedding Planning Tools
1. **Budget Calculator** - Interactive tool for planning and tracking wedding expenses
2. **Wedding Timeline** - Planning timeline with suggested milestones
3. **Wedding Checklist** - Comprehensive task tracking with progress indicators
4. **Wedding Countdown** - Real-time countdown timer to the big day
5. **Mood Board** - Inspiration gallery for design ideas

### Vendor Management & Communication
1. **Vendor Comparison** - Side-by-side vendor comparison tool
2. **Venue Finder** - Interactive venue search with filtering options
3. **Guest Management** - RSVP tracking and guest list management
4. **Real-time Chat** - Communication system between clients and vendors

### Inspiration & Information
1. **Inspiration Gallery** - Categorized wedding inspiration images
2. **Weather Widget** - Wedding date weather forecasts
3. **Features Overview** - Comprehensive feature showcase

### Technical Enhancements
- **Dark Mode** - User preference-based theme switching
- **3D Effects** - Interactive 3D card animations
- **Particle Background** - Dynamic background effects
- **Gradient Text** - Modern text styling effects
- **Enhanced Animations** - Staggered and moving animations

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open your browser to `http://localhost:5173`
